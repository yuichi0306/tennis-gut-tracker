# テニスノート

ガットの張り替え履歴と練習記録をつけ、張り替え時期を管理するテニス用のWebアプリです。

## 機能

- ラケットの登録・編集・削除
- ガット張り替え記録（ガット名、種類、テンション、張った場所、メモ）
- 練習記録（日付、練習時間、メモ）
- ダッシュボードで各ラケットの張り替え時期を自動判定
  - 張り替え後の使用時間・経過日数から「問題なし / そろそろ張り替え時期 / 張り替え推奨」を表示

データはすべてブラウザのlocalStorageに保存されます（サーバー不要）。

## 技術スタック

- React + TypeScript + Vite
- Tailwind CSS
- React Router

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
